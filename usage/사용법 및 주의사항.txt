* 설명
플레이투게더 오토 피시봇입니다.

----------------------------------------------------------------------------------------------------

* 주의사항
해당 봇 매커니즘은 미끼가 물었을시 케릭터 위에 느낌표 이미지 서칭을 통한 자동 낚시 + 키보드 컨트롤을 통한 자동 낚시를 합니다.
고로 몇가지 제약사항이 존재합니다.

- 블루스택이 항상 윈도우 화면 위에 존재한다.
- 블루스택 화면이 좌표 (0 , 0)에 존재한다.
- 대왕 물고기는 잡지 못한다.

----------------------------------------------------------------------------------------------------

* 기능
- 물고기 자동 낚시
- 낚싯대 자동 수리
- 자동 낚시 후 보관하는 KEEP 모드 ( 물고기 상자가 가득찼을시 비우는건 수동입니다 )
- 자동 낚시 후 바로 판매하는 SELL 모드( !!! 즉시 판매 멤버십 가입되있을시에만 정상 작동합니다 !!! )

----------------------------------------------------------------------------------------------------

* 단축키들 
F1 : 오토피시봇 ON OFF
F2 : KEEP 모드로 전환 ( 물고기 잡은후 보관 )
F3 : SEEL 모드로 전환 ( 물고기 잡은후 판매 )
F4 : 종료

----------------------------------------------------------------------------------------------------

사용방법

1. 플레이투게더를 킨다.

2. 다운받은 main.exe 실행 

3. 자동 낚시할 장소로 케릭터를 이동

4. 사용할 낚싯대를 착용한다.

5. F1을 눌러 봇을 가동시킨다.

6. 봇이 낚싯대를 던진다.

7. 미러링된 Enjoy Fish Bot 화면상에서 마우스 움직이면 네모난 커서가 따라다닐텐데 이를 클릭하면 화면에 빨간색 네모박스가 생깁니다.
   이 빨간색 네모박스를 물고기가 미끼를 물었을때 케릭터 위에 느낌표가 뜨는 위치에 클릭하여 빨간색 네모박스를 생성합니다.
   이해가 안간다면 예시이미지들에서 예시1 , 예시2 , 예시3 사진을 참조해주세요.

8. 세팅이 완료되었다면 모든게 끝났습니다. 모드에 따라서 봇이 자동 낚시를 지속합니다.

----------------------------------------------------------------------------------------------------

Tips. 
* 느낌표 인식률 높이는 방법 
- 예시 이미지들처럼 빨간색 박스안은 빈 배경으로 깨끗하게 잡게끔 카메라 앵글을 이동 시켜주면 좋습니다.

낚시터 위치를 이동하였거나 카메라 앵글이 변할때마다 빨간색 박스는 다시 세팅 해주시면됩니다.
